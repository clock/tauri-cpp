# tauri-cpp

Build desktop gui apps in c++ with a Tauri shell and a setup path that does not fight Windows users.

## Main goal

`tauri-cpp` is meant to be the easy path for a standalone desktop app first. It keeps the public c++ layer small, uses a thin Rust bridge with `cxx`, and gives you three frontend starting points without forcing one UI stack on everyone.

## In the repo

- plain html, css, and js example
- Vite + Tailwind example
- React + shadcn/ui example
- Visual Studio solution for common Windows tasks
- GitHub Pages docs built from this repo
