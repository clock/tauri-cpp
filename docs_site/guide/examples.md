# examples

## plain_html

This is the lowest overhead starter. It uses plain html, css, and js, builds by copying static files, and is the best place to start if you care more about setup speed and memory than framework comfort.

## vite_tailwind

This is a normal Vite app with Tailwind and no component library. It is a middle ground between the plain starter and a full React stack.

## react_shadcn

This is the highest level example in the repo. It uses React, Vite, and shadcn/ui style building blocks while keeping the same native demo surface as the other examples.
